# Changelog
 
All notable changes to PPDMStudio will be documented in this file.
 
The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/).
 
---
## [1.3.0] - 2026-05-31

### Added

- **Marker picks viewer** ‚Äî new Marker Picks tab on the Well Header Panel displays stratigraphic picks from the PPDM `STRAT_WELL_SECTION` table
  - Displays marker name, measured depth (ft), TVD (ft), dominant lithology, interpretation ID, and remarks
  - Picks sorted by measured depth
  - Read-only in this release

## [1.2.0] - 2026-05-26

### Added

- **CSV well import** ‚Äî new Import CSV wizard on the Wells page for loading well data from comma-delimited files into the PPDM `WELL` table
  - Three-step wizard: file upload, column mapping, and preview before commit
  - Auto-maps CSV columns whose names match PPDM field names
  - User-defined column mapping for all supported `WELL` fields (UWI, name, operator, field, status, dates, depths, elevations, coordinates, state, county, remark)
  - Duplicate UWIs (matched by UWI) are skipped and reported in the import summary
  - Option to add imported wells directly to the currently open Well List
 
## [1.1.0] - 2026-05-24
 
### Added
 
- **Assigned Field** ‚Äî field is now displayed in the well grid and editable in the well header panel, backed by the PPDM `FIELD` reference table
### Fixed
 
- **County display** ‚Äî well grid now shows the county name instead of the FIPS code
- **MudBlazor analyzer warnings** ‚Äî replaced illegal `Title` attributes on `MudIconButton` with `MudTooltip` wrappers across `WellHeaderPanel`, `WellLocationTab`, and `ReferenceSelect`
- **MudBlazor analyzer warning** ‚Äî removed illegal `PanelClass` attribute from `MudTabs`; padding moved to individual `MudTabPanel` elements
- **Null reference warning** ‚Äî added null guard on `args.Item` in `OnRowClick`

## [1.0.0] - 2026-05-23
 
Initial release.
 
### Added
 
- **Project management** ‚Äî connect to and switch between multiple PPDM SQL Server databases
- **Well grid** ‚Äî search and filter wells, multi-select, and save results as named well lists
- **Well header editing** ‚Äî tabbed panel for identification, dates, depths, elevations, and remarks
- **Location management** ‚Äî cascading state and county autocomplete backed by the PPDM Area hierarchy
- **Reference table management** ‚Äî add new operators, statuses, profile types, states, and counties without leaving the panel
- **Audit trail** ‚Äî all inserts and updates record created/changed user and date
- **Well lists** ‚Äî save, restore, and delete named sets of wells across sessions
