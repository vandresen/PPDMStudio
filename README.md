# PPDM Studio

A modern desktop application for viewing and editing PPDM (Professional 
Petroleum Data Management) data, built with .NET 9, WPF, Blazor Hybrid, 
MudBlazor and Dapper.

## Features
- Project management with secure credential storage
- Well search and filtering
- Well lists — create, save and manage sets of wells
- Built on PPDM 3.9 standard

## Technology Stack
- .NET 9 WPF + Blazor Hybrid
- MudBlazor UI components
- Dapper for data access
- SQLite for local storage
- Windows Credential Manager for secure password storage

## Requirements
- Windows 10/11
- .NET 9
- SQL Server with PPDM database
- Visual Studio 2022

## Getting Started
1. Clone the repository
2. Open `PPDMStudio.sln` in Visual Studio 2022
3. Build and run
4. Create a project pointing to your PPDM database
